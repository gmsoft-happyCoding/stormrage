# install

`npm i stormrage -g`

## sub-commit

`init` 用于根据模板初始化项目

### usage

`npx stormrage init`

> 模板使用了内部的服务器, 请配置你的 hosts 文件:
>
> (stormrage 会尝试自动设置, 如果设置失败, 请手动设置以下规则)
>
> - 192.168.2.11 cdn.gmsoftdev.com
> - 192.168.2.11 registry.gmsoftdev.com
> - 192.168.2.12 registry.gmsofttest.com
